1. 初始化后端
用 Express.js 创建一个简单的后端：

bash
複製程式碼
mkdir backend
cd backend
npm init -y
npm install express body-parser cors

2. 创建后端代码
创建一个 server.js 文件

3. 运行后端
node server.js

测试登录功能
启动前端 (npm run dev) 和后端 (node server.js)。
打开浏览器访问 http://localhost:5173。
使用以下账号登录测试：
Email: user@example.com
